# Sunnyside
This is an exercise where I reproduced a website, following [the design](https://www.figma.com/file/XAkjbnDhAjD4hzHPMmAS4U/sunnyside-becode?type=design&node-id=0-1&mode=design&t=yxJEqAHMSLUobwDi-0) of a Figma. This is a consolidation exercise, the following concepts are used and reviewed:
- Sementic HTML
- SASS + B.E.M methodology
- Grid and Flexbox
- Responsive and mobile first design
- Create a “pixel-perfect” integration from a given wireframe
